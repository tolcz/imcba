• Welcome!

    In this survey you will be asked to record samples of your voice and answer a few simple questions. To maintain quality of the data collected please make sure you work in a relatively quiet environment without external noise.

    Press Next when you are ready to start.

  • Technical Note

    for users of Safari browser

    Users of Apple computers and iPhones reported problems with recording their voice using the default Safari browser. In this case you may:

    1. use one of supported alternative browsers
    2. install a missing plugin and keep the current version of Safari browser, or
    3. upgrade to the latest version of Safari browser
